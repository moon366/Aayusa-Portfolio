================================================================================
                      CUSTOM CURSOR SPECIFICATIONS
================================================================================

Location: About section only
Components:
1. Blue Circle:
   - Size: 256px (w-64 h-64)
   - Color: Semi-transparent blue (bg-blue-400/30)
   - Effect: Solid (no blur)
   - Shadow: 0 0 20px rgba(96, 165, 250, 0.2)

2. Cyan Ring Pointer:
   - Size: 32px (w-8 h-8)
   - Border: 2px solid cyan (border-cyan-400)
   - Shadow: inset 0 0 8px rgba(34, 211, 238, 0.4)
   - Spring: stiffness 150, damping 35, mass 0.8

3. Tracking:
   - Circle: stiffness 700, damping 20, mass 0.2 (tight tracking)
   - Ring: stiffness 150, damping 35, mass 0.8 (smooth follow)
   - Hidden default cursor within About section

================================================================================
                         CHALLENGES & SOLUTIONS
================================================================================

1. Navbar Active Section Not Updating
   Solution: Adjusted IntersectionObserver rootMargin from "-30% 0px -65% 0px"
            to "-50% 0px -50% 0px" for center-of-screen detection

2. Duplicate Export Errors in About.jsx
   Solution: Removed duplicate code and cleaned up malformed JSX fragments

3. Custom Cursor Performance
   Solution: Used fixed positioning with transform (GPU acceleration)
            Implemented proper spring physics with appropriate damping

4. Custom Cursor Scope
   Solution: Moved CustomCursor from global Home to local About component
            Only renders when user is in About section

5. Tailwind Linting Warnings
   Solution: Files have minor linting suggestions (gradle vs gradient naming)
            These are preferences, not errors - both work correctly
